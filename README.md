# Mentora

**Mentora: A Context-Aware LLM-Based Faculty Teaching Assistant**  
Capstone Project – Lorma Colleges – College of Computer Studies and Engineering  

---

## 🚀 Project Overview

Mentora is a **web-based, AI-powered faculty assistant** designed to help instructors manage courses, generate assessments, and monitor student performance efficiently. Using **context-aware large language models (LLMs)** and automated grading, Mentora reduces administrative workload and provides actionable analytics for faculty members.

Key features include:

- Faculty login and course management
- Upload syllabus and course materials
- Automatic quiz and exam generation from uploaded materials
- AI-assisted grading for objective and subjective questions
- Basic predictive analytics for identifying at-risk students
- Interactive dashboard with performance visualizations
- Context-aware AI chat assistant for faculty queries

Mentora focuses on **quality education** and supports **SDG 4 (Quality Education)** and **SDG 9 (Innovation and Infrastructure)**.

---

## 🧩 Features

### 1. Course & Material Management
- Create, view, and manage courses
- Upload PDF syllabus and materials
- Automatic extraction of text for AI processing

### 2. Assessment Automation
- Generate quizzes/exams from syllabus content
- Auto-grade multiple-choice and short-answer questions

### 3. Student Analytics
- Visualize student grades and class performance
- Identify at-risk students
- Highlight top-performing students

### 4. AI Faculty Assistant
- Chat with a context-aware AI powered by LLM
- Ask for quiz generation, summaries, or intervention suggestions
- AI responses are tailored to uploaded course materials

## 🛠️ Technology Stack

- **Backend:** Laravel 12 (PHP)  
- **Frontend:** Blade with Alpine.js  
- **Database:** MySQL / MariaDB  
- **AI Integration:** OpenAI GPT API  
- **Charts:** Chart.js  
- **File Parsing:** smalot/pdfparser (PDF text extraction)
